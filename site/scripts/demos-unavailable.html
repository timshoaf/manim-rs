The wasm bundle for this deploy failed to produce, so the live demo is not available at this URL. This page exists so the links pointing here are honest rather than a 404 — the rest of the site built normally.

The demo build is deliberately non-blocking in CI: it bundles through wasm-bindgen and wasm-opt, and a failure there should not take down the book and the API reference.

Run it locally

git clone https://github.com/cryptex-ai/manim_rust
cd manim_rust/examples/dioxus-app

rustup target add wasm32-unknown-unknown
cargo install dioxus-cli

dx serve --platform web    # http://localhost:8080
